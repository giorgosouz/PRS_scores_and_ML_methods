# PRS Scores and ML Methods

Diploma thesis code for preprocessing openSNP / PRS data and running classical machine learning experiments.

## Repository contents
- `opensnp_data_take_inds.py`: data extraction and indexing helper
- `raw_to_npy.py`: raw-to-NumPy preprocessing step
- `ml.py`: main machine learning workflow
- `classifiers.py`: classifier definitions and helpers
- `diplwmatikh_ouzounidis.pdf`: thesis document

## Notes
- The repository captures research workflow code rather than a standalone end-user application.
- The thesis PDF is the best entry point for methodology and problem framing.
